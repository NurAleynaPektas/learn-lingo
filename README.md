# LearnLingo

A modern language learning platform where users can explore tutors, filter them based on preferences, and save favorites.

🚀 **Live Demo:** https://learn-lingo-brown.vercel.app

---

## Features

- **Authentication (Firebase)**
  - Register & Login
  - Persistent user session

- **Teachers Listing**
  - Dynamic teacher cards
  - Load more functionality

- **Favorites System**
  - Add/remove favorites
  - User-specific favorites (per Firebase UID)
  - Stored in localStorage

-  **Advanced Filters**
  - Filter by language
  - Filter by level
  - Filter by price range

-  **Smooth UX**
  - Global loader (login & filtering)
  - Toast notifications (iziToast)
  - Responsive design

-  **Mobile-first UI**
  - Fully responsive layout
  - Modern card & header design

---

##  Tech Stack

- **Frontend:** React (Vite)
- **Routing:** React Router
- **Forms:** React Hook Form + Yup
- **Auth:** Firebase Authentication
- **Styling:** CSS Modules
- **Notifications:** iziToast
